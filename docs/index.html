Technical Architecture

ViewModel State Management

The app relies on MainScreenViewModel which exposes a single, read-only GameState flow. Redux-style state flows prevent race conditions and ensure UI consistency.

State Freezing Overlay

Features state-freezing hooks inside the main Compose loop. When a round ends, details are locked locally. This prevents the next round's word from blinking on the card during the dialog fade transition.

Modern Jetpack Nav 3

Uses the latest androidx.navigation3 framework to organize screens and activities using type-safe serialization keys.

Getting Started

  1. Prerequisites

    Ensure you have the latest stable version of Android Studio installed.

  2. Clone / Locate Files

    Locate the project folder inside your workspace workspace directory: c:\Users\lawli\OneDrive\Desktop\antig.

  3. Open in Android Studio

    Select File > Open inside Android Studio, navigate to the folder, and click OK. Android Studio will automatically build the project dependencies.

  4. Build & Run

    Run the project on a virtual device emulator (AVD) or a connected phone by clicking the green Run button in the toolbar.
